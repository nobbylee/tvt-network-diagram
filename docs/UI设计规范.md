---

## 1. 设计方向

### 1.1 风格定位（已确认）

**浅色简洁风（默认）** — 参考 Figma / Linear / draw.io 的编辑器体验：

- 干净、克制，长时间使用不疲劳
- 画布区域最大化，侧边栏紧凑
- TVT 品牌蓝 `#0066CC` 作为强调色
- 深色主题保留为可选切换，非默认

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 简洁 | 去掉多余装饰和按钮，属性修改即生效（无需「应用更改」） |
| 画布优先 | 画布占屏幕 60%+，元件库和属性面板可折叠（后续） |
| 按需显示 | 节点上仅展示已填写的信息，空字段不占位 |
| 品牌可辨 | TVT 蓝、第三方/通用灰，顶部细色条 + 图标底色 |
| 低学习成本 | 拖拽即用，选中后在右侧编辑属性 |

## 2. 色彩系统

### 2.1 品牌色

| 用途 | 色值 | 说明 |
|------|------|------|
| TVT 主色 | `#0066CC` | 按钮、选中态、连线 |
| TVT 悬停 | `#0052A3` | 按钮 hover |
| TVT 浅底 | `#EFF6FF` | 选中区域、hover 背景 |
| 第三方 / 通用 | `#64748B` | 第三方设备、交换机等 |

### 2.2 界面色（浅色主题 · 默认）

| 令牌 | 色值 | 用途 |
|------|------|------|
| `--app-bg` | `#F1F5F9` | 应用外背景 |
| `--panel-bg` | `#FFFFFF` | 侧边栏、工具栏 |
| `--canvas-bg` | `#F8FAFC` | 画布背景 |
| `--canvas-grid` | `#CBD5E1` | 点阵网格 |
| `--text-primary` | `#0F172A` | 主文字 |
| `--text-secondary` | `#64748B` | 次要文字 |
| `--panel-border` | `#E2E8F0` | 边框、分割线 |

### 2.3 界面色（深色主题 · 可选）

工具栏右上角可切换，非默认。色值见 `packages/web/src/index.css`。

## 3. 布局规范

### 3.1 整体布局

```
┌──────────────────────────────────────────────────────────────┐
│  工具栏  h=48px                                               │
├──────────┬───────────────────────────────────┬───────────────┤
│ 元件库    │         画布区域                   │   属性面板     │
│ w=224px  │         flex-1                    │   w=256px     │
└──────────┴───────────────────────────────────┴───────────────┘
```

### 3.2 尺寸规范

| 元素 | 尺寸 |
|------|------|
| 工具栏高度 | 48px |
| 元件库宽度 | 224px |
| 属性面板宽度 | 256px |
| 设备节点宽度 | 160px（有额外字段时自适应增高） |
| 圆角 | 面板 8px，节点 8px，按钮 6px |
| 字号 | 标题 14px，正文 12px，辅助 10px |

## 4. 组件规范

### 4.1 设备节点（画布上）

节点信息 **名称 + 型号 + IP 均为选填**，但 **名称建议必填**（拖入时自动生成默认名）。画布上 **只显示已填写的字段**：

**完整信息：**
```
┌──────────────────────┐
│ █ 品牌色条            │
│ [图标]  设备名称       │
│         型号           │
│  192.168.1.10        │
└──────────────────────┘
```

**仅名称：**
```
┌──────────────────────┐
│ █ 品牌色条            │
│ [图标]  NVMS 平台      │
└──────────────────────┘
```

**名称 + IP（无型号）：**
```
┌──────────────────────┐
│ █ 品牌色条            │
│ [图标]  核心交换机     │
│  192.168.1.1         │
└──────────────────────┘
```

- **选中态：** 蓝色 2px 边框 + 浅蓝外发光
- **IP 地址：** 等宽字体，独立一行灰底标签

### 4.2 属性面板字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 设备名称 | 建议填 | 拖入时自动生成，可改 |
| 品牌 | 自动 | 由元件库决定，可改 |
| 型号 | 选填 | 手动输入 |
| IP 地址 | 选填 | 手动输入 |
| 备注 | 选填 | 不在节点上显示 |
| 连线端口 | 选填 | 选中连线时显示 |
| 连接类型 | 选填 | 网线 / 光纤 / PoE 等 |

属性修改 **即时反映到节点**，无需确认按钮。

### 4.3 元件库卡片

- 左侧 32×32 图标区，品牌色浅底
- 右侧设备类型名称
- Hover：边框变蓝 + 浅蓝背景
- 可拖拽

### 4.4 连线

- 颜色：`--accent`，60% 透明度，线宽 2px
- 标签：仅在有连接类型等信息时显示

---

## 5. 图标风格

- 线性 SVG（stroke 1.5px），跟随品牌色
- 简洁几何形状，不追求写实
- 按形态分类，型号不靠图标区分

## 7. 前端网页预览

已实现可预览的前端网页（浅色简洁基调）：

| 页面 | 内容 |
|------|------|
| **项目列表** | 搜索、新建、项目卡片（客户名 / 设备数 / 更新时间） |
| **编辑器** | 工具栏 + 元件库 + 画布示例拓扑 + 属性面板（可点击节点即时编辑） |

```bash
cd packages/web
npm run dev
```

浏览器打开 `http://localhost:5173`。

**可体验：**
1. 首页浏览项目列表 → 点击「XX银行监控项目」进入编辑器
2. 点击画布节点 → 右侧属性面板联动，修改名称/型号/IP 即时反映到节点
3. 左上角返回按钮回到项目列表
